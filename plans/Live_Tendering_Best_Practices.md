# Live Tendering Best Practices

## Pre-Auction Setup:
1. Vendor pre-qualification (min 3 vendors)
2. Clear auction rules publication 
3. Test bidding environment 

## During Auction:
⚠️ Real-time bid visibility rules:
- Show ranking but hide identities 
- Automatic extension if bids in last 2min 

## Post-Auction:
- Immediate provisional award 
- Audit log generation
