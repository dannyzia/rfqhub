# Limited Tendering Vendor Invitation Guide

## Invitation Workflow:
1. Create vendor shortlist (from registered pool)
2. System sends encrypted invitations 
3. Vendors confirm participation 

## Validation Rules:
❗ Minimum invited vendors: 3 for goods, 5 for works  
❗ Maximum invitation period: 7 days  
❗ Mandatory justification memo  

## Emergency Procurement:
Special rules for urgent cases (<72hr process)
