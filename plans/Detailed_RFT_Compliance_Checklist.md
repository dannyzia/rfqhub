# Detailed RFT Compliance Checklist

## Mandatory Sections:
✅ Bangladesh tender type validation (PG/PW/PPS)
✅ Bid security calculation  
✅ Complete document package:
   - Technical specifications 
   - BoQ in prescribed format
   - Evaluation methodology

## Type-Specific Requirements:
### PG1 (Goods <$500k):
- 2 bid opening stages required
- Sample submission mandatory

### PW3 (Works >$5M):
- Site visit documentation 
- Subcontractor approval process
